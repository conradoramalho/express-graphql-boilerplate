# express-graphql-boilerplate

This is a simple graphql boilerplate

## Features

- Express server
- GraphQL
- Mongoose
- Eslint
- Babel 7
- Docker

## Installation

```
  yarn install
```

## Usage

### Dev

Start the server

```
  yarn start
```

GraphiQL will be running at http://localhost:3000/graphiql/

### Production

Build the server

```
  yarn build
```

Run the server

```
  yarn production
```

The server will be running at http://localhost:3000
