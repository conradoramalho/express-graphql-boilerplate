# express-graphql-boilerplate

This is a simple graphql boilerplate

## Features

* Express server
* GraphQL
* Mongoose
* Eslint
* Babel 7

## Installation

```
yarn install
```

## Usage

### Production

Build the server

```
yarn build
```

Run the server

```
yarn production
```

The server will be running at http://localhost:3010/graphql/

### Dev

Start the server

```
yarn start
```

GraphiQL will be running at http://localhost:4000/graphiql/
